## Why CEFileManager | 为什么是 CEFileManager

- **Android 上直接运行终端工作流** / Run real terminal workflows directly on Android
- **不止一两个工具，而是 12 个 CLI 接入** / 12 integrated CLI tools instead of just one or two
- **支持免费模型与托管模型在线使用** / Online access to free and managed models
- **APK、XML、DEX、SO、压缩包、二进制工具集中在一个 App** / APK, XML, DEX, SO, archive, and binary workflows in one app
- **Root、Shizuku、SAF、内部存储、受限目录统一处理** / Unified handling for Root, Shizuku, SAF, internal storage, and restricted paths
- **文件管理、编辑器、终端、签名、导出一套工作流完成** / File manager, editor, terminal, signing, and export in one workspace

## Positioning | 产品定位

**中文**

CEFileManager 不是普通文件浏览器，而是面向高阶 Android 用户的移动工程工作站。  
它把终端、AI CLI、在线模型、文件工程、APK 修改、逆向工具和深度存储访问整合到同一个 App 里，让手机真正具备可连续作业的能力。

**English**

CEFileManager is not just a file browser.  
It is a mobile engineering workstation for advanced Android users, combining terminal workflows, AI CLI access, online models, package editing, reverse-oriented tools, and deep storage access in one app.

## Online Models | 在线模型

**手机端直连免费模型与托管模型。**  
**Direct access to free models and managed hosted models on phone.**

在线模型能力不只是一条固定线路，而是支持多种接入方式与实时下发模型配置。

Online model access is not limited to one hardcoded path. The app supports multiple access modes and real-time delivered model profiles.

- **支持 FREE 与托管模型目录** / Supports FREE and managed model catalogs
- **支持移动端切换当前可用模型** / Supports model switching on mobile
- **支持 OpenAI-compatible、Anthropic-compatible、Google、Qwen、DeepSeek、OpenRouter 等上游** / Supports OpenAI-compatible, Anthropic-compatible, Google, Qwen, DeepSeek, OpenRouter, and other routed upstreams
- **支持 API Key、OAuth、服务器托管代理会话** / Supports API key, OAuth, and server-managed proxy sessions
- **支持根据服务器实时分发结果使用免费线路与托管线路** / Supports free lines and managed lines according to real-time server distribution

## Runtime & Dependencies | 运行时与依赖

CEFileManager provides a mobile runtime environment for CLI and tooling workflows.

- **内置 Linux 运行时** / Embedded Linux runtime
- **Node.js / npm CLI 安装链** / Node.js / npm CLI installation flow

- **CLI 托管配置下发** / Managed CLI configuration delivery
- **MCP 相关工作流支持** / MCP-related workflow support
- **本地回环代理链路** / Local loopback proxy routing

## Reverse & Package Toolkit | 逆向与包体工具

- **APK 打开、编辑、重打包、签名、导出** / Open, edit, repack, sign, and export APK packages
- **XML 编辑与资源工作流** / XML editing and resource workflows
- **DEX 分析与修改能力** / DEX-oriented analysis and modification
- **SO / ELF 二进制分析** / SO / ELF binary-oriented analysis
- **虚拟压缩包浏览与归档操作** / Virtual archive browsing and archive operations
- **Hex 查看、哈希校验、结构化包体检查** / Hex viewing, hash checks, and structured package inspection

项目代码中也包含 `Jadx`、`Baksmali`、签名处理与原生分析工具链相关集成。

The codebase also includes integration around `Jadx`, `Baksmali`, signing flows, and native analysis utilities.
